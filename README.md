# Zepto SQL Data Analysis Project

## Overview
Analyzed Zepto product data using SQL to uncover insights related to pricing, discounts, inventory, and category performance.

## Tools Used
- SQL
- PostgreSQL

## Key Highlights
- Analyzed product pricing, discount patterns, inventory levels, and category-wise performance.
- Used Aggregate Functions, GROUP BY, HAVING, CASE Statements, Window Functions, and Subqueries to answer business questions.
